# Notes App (Nuxt 4 SPA)

Тестовое задание: SPA приложение для управления заметками.

## Стек

- Nuxt 4 (Composition API)
- Pinia (state management)
- SCSS
- pnpm
- Docker

---

## Описание

Приложение для создания и редактирования заметок.

Каждая заметка содержит:
- Заголовок
- Список задач (Todo)

Каждый Todo:
- Чекбокс (выполнено / не выполнено)
- Текст задачи

---

## Страницы

### 1 Главная страница

Отображает список всех заметок.

Доступные действия:
- Создание новой заметки
- Переход к редактированию
- Удаление заметки (через модальное окно)

Todo на главной отображаются сокращённо и без возможности редактирования.

---

### 2 Страница редактирования

Позволяет:
- Изменять заголовок
- Добавлять Todo
- Удалять Todo
- Редактировать текст Todo
- Отмечать как выполненные
- Сохранять изменения
- Отменять редактирование (через модальное окно)
- Удалять заметку (через модальное окно)
- Undo (Ctrl + Z)
- Redo (Ctrl + Shift + Z)

---

## Undo / Redo

Реализован механизм истории изменений:
- Отмена последнего действия
- Повтор отменённого действия
- Работает для всех изменений заметки и Todo

---

## Сохранение данных

Состояние приложения синхронизируется с `localStorage`.

После перезагрузки страницы данные сохраняются.
